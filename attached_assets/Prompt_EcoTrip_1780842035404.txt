Crie uma aplicação Full Stack chamada:

"EcoTrip Arboretum – Simulador de Impacto Ambiental e Compensação Florestal"

OBJETIVO

Desenvolver uma plataforma web que permita ao usuário calcular a emissão de CO₂ de uma viagem, comparar diferentes meios de transporte e visualizar como compensar esse impacto através da restauração florestal inspirada no Programa Arboretum.

O sistema deve possuir caráter educativo, ambiental e demonstrar boas práticas de desenvolvimento assistido por IA.

--------------------------------------------------

TECNOLOGIAS

Frontend:
- React
- JavaScript (não utilizar TypeScript)
- Tailwind CSS
- Recharts para gráficos

Backend:
- Node.js
- Express

Banco:
- SQLite

Testes:
- Jest
- React Testing Library

FUNCIONALIDADE 3
CÁLCULO DE EMISSÃO

Utilizar fatores médios:

Caminhada = 0

Bicicleta = 0

Metrô = 0.02

Trem = 0.03

Ônibus = 0.09

Motocicleta = 0.11

Carro Gasolina = 0.12

Carro Etanol = 0.06

Carro Elétrico = 0.02

Avião = 0.25

Fórmula:

Emissão CO₂ = Distância × Fator de Emissão

Resultado em:

kg CO₂

FUNCIONALIDADE 6
COMPENSAÇÃO FLORESTAL

Diferencial inspirado no Programa Arboretum.

Calcular:

Quantidade estimada de mudas necessárias para compensar a emissão.

Considerar:

1 muda nativa = 15 kg CO₂ compensados ao longo do crescimento.

Exemplo:

Emissão:
60 kg CO₂

Resultado:

São necessárias aproximadamente 4 mudas nativas para compensar essa emissão.

Exibir:

🌳 Quantidade de mudas
🌳 Área estimada de restauração

Considerar:

1 muda = 4 m² restaurados
